Instructions for Running the Server-Client Pokemon Search and Save Program:

Step 1:
Open two terminals and navigate to the directory containing the source files.

Step 2:
In Terminal 1, enter the following commands:
    make all
    ./server

Step 3:
The server will prompt you to enter the name of the pokemon.csv file. Provide the filename containing Pokemon descriptions when prompted.

Step 4:
In Terminal 2, enter the following command:
    ./client

Step 5:
The client will display a menu with options. Choose any of the options(1/2/3). 

Step 6:
After you have finished using the program, exit the client by choosing option 3 from the menu.
In Terminal 1, enter the following command to clean up the compiled files:
    make clean
